## Manual Monitoring Checklist

### Weekly Manual Checks

#### Website Review (Each Competitor)
- [ ] Homepage changes (messaging, offers, imagery)
- [ ] New pages or services added
- [ ] Pricing page updates
- [ ] Portfolio/gallery additions
- [ ] Blog post activity
- [ ] Contact/location changes
- [ ] Trust signals (awards, certifications, reviews)

#### Social Media Audit
- [ ] Facebook: Posts, engagement, ads, reviews
- [ ] Instagram: Posts, stories, reels, follower growth
- [ ] YouTube: New videos, view counts, subscriber changes
- [ ] Pinterest: New boards/pins, engagement
- [ ] LinkedIn: Company updates, job posts, articles
- [ ] TikTok: Content style, engagement, trends

#### Local Presence Check
- [ ] Google Business Profile updates
- [ ] New photos/posts on GMB
- [ ] Review responses (style, speed)
- [ ] Service area changes
- [ ] New Q&A activity

#### Advertising Observation
- [ ] Google search ads (search competitor + location)
- [ ] Facebook Ads Library check
- [ ] YouTube pre-roll ads
- [ ] Display ads on relevant sites
- [ ] Local print/radio/TV if applicable

#### Review Monitoring
- [ ] Google Reviews: New reviews, responses, rating changes
- [ ] Yelp: Activity and responses
- [ ] BBB: Complaints and ratings
- [ ] Angi/HomeAdvisor: Reviews and badges
- [ ] Houzz: Projects and reviews

## Alert Response Protocol

### High-Priority Alerts (Respond within 24 hours)
- Competitor launches major promotion
- Negative PR about competitor (opportunity)
- Competitor website down (increased ad spend opportunity)
- New competitor enters market
- Competitor wins major award

### Medium-Priority Alerts (Respond within 1 week)
- Competitor publishes relevant content
- New keyword rankings achieved
- Social media campaign launch
- Review volume spike

### Low-Priority Alerts (Monthly review)
- Minor website updates
- Regular content publishing
- Gradual metric changes
