## Content Decay Identification

### What is Content Decay?

Content decay occurs when previously high-performing content loses traffic and rankings over time. Common causes:
- Outdated information
- Increased competition
- Algorithm changes
- Seasonal fluctuations
- Technical issues

### Content Decay Monitoring

#### Decay Detection Thresholds

| Metric | Warning Threshold | Critical Threshold |
|--------|-------------------|-------------------|
| Traffic drop (3mo) | -20% | -40% |
| Ranking drop | 5+ positions | 10+ positions |
| Impressions drop | -25% | -50% |
| CTR drop | -2% absolute | -5% absolute |

### Decay Recovery Actions

1. **Content Freshness Update**
   - Update statistics and data
   - Add new sections
   - Refresh examples
   - Update publish date

2. **SEO Re-optimization**
   - Research current SERP
   - Update title/meta
   - Improve keyword targeting
   - Add new relevant keywords

3. **Content Expansion**
   - Add FAQs
   - Include new topics
   - Add visuals
   - Increase word count

4. **Technical Fixes**
   - Check page speed
   - Fix broken links
   - Update internal links
   - Improve mobile experience

## Tools for Content Performance Tracking

### Primary Tools

| Tool | Purpose | Frequency |
|------|---------|-----------|
| Google Analytics 4 | Traffic, engagement, conversions | Daily |
| Google Search Console | Impressions, clicks, position | Daily |
| SEMrush/Ahrefs | Keyword rankings, content audit | Weekly |
| Screaming Frog | Technical content audit | Monthly |

### Automated Reporting Setup

1. **GA4 Custom Reports**
   - Content performance by page
   - Blog performance overview
   - Conversion by content type

2. **GSC Data Export**
   - Weekly query performance
   - Page impressions/clicks
   - Position tracking

3. **Looker Studio Dashboard**
   - Combined GA4 + GSC data
   - Content decay alerts
   - Performance trends

### Content Tracking Spreadsheet

Create Google Sheet with tabs:
1. **Master Content List** - All content with metadata
2. **Monthly Performance** - Monthly metrics by content
3. **Decay Tracking** - 3-month comparison
4. **Update Log** - Record of all updates
5. **Gap Analysis** - Content opportunities
6. **Editorial Calendar** - Planned content
