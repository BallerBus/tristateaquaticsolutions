## Implementation Instructions

### For Gmail
1. Go to Settings > See all settings > General
2. Scroll to "Signature"
3. Create new signature
4. Paste HTML code (use "Insert as HTML" or copy from rendered preview)
5. Save changes

### For Outlook
1. File > Options > Mail > Signatures
2. Create new signature
3. Paste HTML code
4. Set as default for new messages and replies

### For Apple Mail
1. Mail > Preferences > Signatures
2. Create new signature
3. Uncheck "Always match my default message font"
4. Paste HTML content

### For Mobile Devices
Use the simplified mobile-friendly versions which render better on small screens and have tap-to-call/email functionality.
