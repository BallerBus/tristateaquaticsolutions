## 5. General Document Specifications

### Margins and Spacing
| Element | Specification |
|---------|---------------|
| Top margin | 0.75" |
| Bottom margin | 0.75" |
| Left margin | 1" |
| Right margin | 1" |
| Paragraph spacing | 12pt after |
| Line spacing | 1.5 (body text) |
| Header height | 1.25" - 1.5" |
| Footer height | 0.5" - 0.75" |

### Typography Hierarchy
```
H1 - Document Title
    Font: Montserrat Bold
    Size: 18pt
    Color: Deep Navy (#0E1F34)
    Case: Title Case or UPPERCASE

H2 - Section Headers
    Font: Montserrat SemiBold
    Size: 14pt
    Color: Deep Navy (#0E1F34)
    Case: Title Case

H3 - Subsection Headers
    Font: Open Sans SemiBold
    Size: 12pt
    Color: Aqua Blue (#00A6B2)
    Case: Title Case

Body Text
    Font: Open Sans Regular
    Size: 11pt
    Color: #333333
    Line Height: 1.5

Captions/Notes
    Font: Open Sans Regular
    Size: 9pt
    Color: #666666
    Style: Italic for notes

Footer Text
    Font: Open Sans Regular
    Size: 8pt
    Color: #888888 or White (on navy)
```

### Design Guidelines

#### Logo Usage
- Minimum clear space: Height of logo on all sides
- Minimum size: 60px width (digital), 0.75" (print)
- Primary version: Color on white/light backgrounds
- Reverse version: White on navy/dark backgrounds
- Never stretch, rotate, or alter proportions

#### Color Application
- Navy (#0E1F34): Headers, primary text, footer backgrounds
- Aqua (#00A6B2): Accents, links, highlights, subheaders
- Gold (#C9A962): Premium touches, decorative elements, taglines
- Gray (#F5F7FA): Backgrounds, info boxes, table alternates
- White (#FFFFFF): Body background, reverse text

#### Graphical Elements
- Horizontal rules: 1-2pt, aqua or gold
- Box borders: 1pt gray (#DDDDDD) or none with shadow
- Corner radius: 4-8px for boxes and buttons
- Icons: Line style, aqua or navy, consistent weight
