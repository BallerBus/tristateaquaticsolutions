### 7.2 Common Citation Issues to Fix

| Issue | How to Identify | How to Fix |
|-------|-----------------|------------|
| **Duplicate Listings** | Multiple entries for same business | Claim and merge, or request removal |
| **Wrong Phone Number** | Old number, typo, area code issue | Update or claim listing |
| **Wrong Address** | Old location, partial address | Update with proof of address |
| **Inconsistent Name** | "Tri State" vs "Tri-State" vs LLC | Standardize to primary NAP |
| **Missing Listing** | Not found in directory | Create new listing |
| **Wrong Category** | Listed as "General Contractor" only | Update to Pool Contractor |
| **Closed/Out of Business** | Marked as permanently closed | Claim and reopen |
| **Hijacked Listing** | Someone else claimed your business | Dispute and reclaim |

### 7.3 How to Fix Inconsistencies

**For Unclaimed Listings:**
1. Go through the claim process
2. Verify ownership
3. Update all information to match primary NAP
4. Add photos, hours, description

**For Claimed Listings (someone else's):**
1. Look for "Suggest an Edit" or "Report" option
2. Provide proof of ownership
3. Contact directory support directly
4. Submit legal business documents if required

**For Duplicate Listings:**
1. Determine which listing is most complete/has reviews
2. Claim the better listing
3. Report the duplicate for removal
4. Some directories have merge features

### 8.3 Review Management Protocol

**Positive Reviews (4-5 stars):**
1. Respond within 24-48 hours
2. Thank the customer by name
3. Reference specific details from their review
4. Invite them back / offer continued service

**Negative Reviews (1-3 stars):**
1. Respond within 24 hours
2. Apologize for their experience
3. Take the conversation offline (provide phone/email)
4. Never be defensive or argumentative
5. Follow up and try to resolve
6. Document the issue internally

**Fake/Spam Reviews:**
1. Document evidence (screenshot)
2. Report to platform using proper process
