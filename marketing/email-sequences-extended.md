# Implementation Guide

## Platform Requirements

**Recommended platforms:** ActiveCampaign, HubSpot, or Mailchimp

**Essential features:**
- Conditional logic and branching
- Date-based triggers (anniversaries, seasons)
- Lead scoring integration
- CRM synchronization
- A/B testing capability

## Sequence Priority Rules

When a contact qualifies for multiple sequences:

1. **Post-Quote Follow-up** takes priority over all others
2. **Active nurture sequences** pause for seasonal campaigns
3. **VIP sequences** layer on top of standard sequences
4. **Referral requests** wait until post-installation sequences complete
5. **Re-engagement** only triggers after all other sequences have completed/exited

## Exit Conditions

| Sequence | Exit When |
|----------|-----------|
| Lost Lead Re-engagement | Replies, schedules consultation, or completes sequence |
| Post-Quote Follow-up | Signs contract or explicitly declines |
| Seasonal Campaigns | Books consultation or sequence completes |
| Fall Planning | Books consultation or October 31 passes |
| Referral Request | Provides referral or sequence completes |
| Anniversary | Completes or requests do-not-contact |
| VIP | Manual removal only |
| Event Invitation | RSVPs or event date passes |

## Sending Windows

**Best days:** Tuesday, Wednesday, Thursday
**Best times:** 9:00-11:00 AM or 2:00-4:00 PM
**Avoid:** Mondays (inbox overload), Fridays (weekend mindset), weekends

## A/B Testing Protocol

**Minimum sample:** 50 sends per variant
**Test duration:** 2 weeks per test
**Primary metric:** Open rate for subject lines, click rate for content
**Secondary metric:** Reply rate

## Personalization Requirements

**Required fields:**
- First name
- Email
- Location/neighborhood
- Lead source

**Enhanced personalization:**
- Pool type interest
- Quote amount (for post-quote)
- Installation date (for anniversary/VIP)
- Referral code (for referral sequences)
- Years owned (for anniversary)

## Compliance Checklist

- [ ] Unsubscribe link in every email
- [ ] Physical business address included
- [ ] Reply-to address monitored
- [ ] Unsubscribe honored within 48 hours
- [ ] CAN-SPAM compliant content
- [ ] No misleading subject lines
