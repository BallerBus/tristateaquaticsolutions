## How to Use These Materials

### For Cold Outreach
1. Start with Letter Template #1 (Initial Introduction)
2. Follow up with Letter Template #2 if no response in 2 weeks
3. If interest is shown, send the Information Packet

### For Presentation Opportunities
1. Use Letter Template #3 to request board meeting slot
2. Present using the Presentation Deck
3. Leave behind printed Information Packet + Discount Structure
4. Send Email #1 (Post-Presentation Thank You) within 24 hours

### For Active Partnerships
1. Send Email #2 (Resident Communication Template) for HOA to distribute
2. Send Email #3 (Seasonal Check-In) quarterly or at start of pool season
3. Send Email #4 (Annual Renewal) 30 days before partnership anniversary
