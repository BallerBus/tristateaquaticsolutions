## 5. Rich Pin Setup Instructions

Rich Pins automatically sync information from your website to your pins, showing real-time pricing, availability, and descriptions. For a pool business, **Article Rich Pins** are most relevant for blog content.

### Step 1: Add Meta Tags to Website

Add Open Graph meta tags to your website's HTML `<head>` section. For each page that should generate Rich Pins:

```html
<!-- Required Open Graph Tags -->
<meta property="og:type" content="article" />
<meta property="og:title" content="Page Title Here" />
<meta property="og:description" content="Page description here (160 chars recommended)" />
<meta property="og:url" content="https://tristateaquaticsolutions.com/page-url/" />
<meta property="og:image" content="https://tristateaquaticsolutions.com/images/featured-image.jpg" />
<meta property="og:site_name" content="Tri-State Aquatic Solutions" />

<!-- Recommended Additional Tags -->
<meta property="article:author" content="Tri-State Aquatic Solutions" />
<meta property="article:published_time" content="2026-01-15" />
```

### Step 2: Validate Your Rich Pins

1. Go to Pinterest Rich Pin Validator: https://developers.pinterest.com/tools/url-debugger/
2. Enter a URL from your website (e.g., a blog post URL)
3. Click "Validate"
4. If successful, you'll see a preview of how your Rich Pin will appear
5. Click "Apply" to enable Rich Pins for your entire domain

### Step 3: Common Issues & Fixes

| Issue | Solution |
|-------|----------|
| "No meta tags found" | Ensure OG tags are in `<head>`, not `<body>` |
| Image not showing | Image must be at least 600x315px, use absolute URL |
| Description truncated | Keep descriptions under 160 characters |
| Validator timeout | Check website hosting speed, try again |

### Step 4: Verify Implementation

After applying for Rich Pins:
- Approval typically takes 24-48 hours
- Pinterest will email confirmation
- Test by creating a new pin from your website
- Rich Pin indicator will appear on the pin

### Rich Pin Best Practices

1. **Every blog post** should have unique OG tags
2. **Update OG images** when refreshing content
3. **Use compelling descriptions** - they show on the pin
4. **Include your brand name** in og:site_name
5. **Test after website updates** to ensure tags weren't removed
