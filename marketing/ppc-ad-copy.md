## AD PERFORMANCE NOTES

### Google Ads Best Practices:
- A/B test headlines regularly
- Use responsive search ads for automatic optimization
- Implement conversion tracking on form submissions and calls
- Set up remarketing lists from website visitors
- Use bid adjustments for high-performing locations/times
- Negative keyword list: DIY, above ground, inflatable, rental, repair (unless for maintenance campaign)

### Facebook/Instagram Best Practices:
- Use high-quality pool imagery and video
- Test carousel ads with multiple pool styles
- Create video testimonials from happy customers
- Use Facebook Pixel for conversion tracking
- Implement Conversions API for iOS 14.5+ accuracy
- Refresh creative every 2-3 weeks to combat ad fatigue
- Test different CTA buttons per objective

### Budget Allocation Recommendation:
- Brand Campaign: 10%
- Geographic Campaigns (PA/DE): 25%
- Cost/Financing Queries: 15%
- Local/Main Line: 15%
- Seasonal Urgency: 15%
- Conquest/Competitive: 10%
- Other Campaigns: 10%

### Key Metrics to Track:
- Cost per lead (CPL)
- Cost per consultation booked
- Lead to consultation rate
- Consultation to sale rate
- Return on ad spend (ROAS)
- Click-through rate (CTR)
- Quality Score (Google)
- Relevance Score (Facebook)
