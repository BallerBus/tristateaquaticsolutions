## Design Guidelines for Vehicle Wrap

**LAYOUT PRIORITIES:**
1. Phone number - largest, most visible element (readable from 50+ feet)
2. Website - second priority (readable from 30+ feet)
3. "Premium Fiberglass Pools" - brand message
4. "2-4 Weeks" - key differentiator
5. Service area - Main Line PA & Northern Delaware

**COLOR SCHEME:**
- Primary background: Deep Navy (#0E1F34)
- Accent elements: Aqua Blue (#00A6B2)
- Text: White (for contrast)
- Optional: Pool imagery with blue water tones

**DO:**
- Keep text readable at highway speeds
- Use high-contrast colors
- Include one strong visual element
- Make phone number prominently visible
- Keep design clean and uncluttered

**DON'T:**
- Overcrowd with too much text
- Use more than 2-3 colors
- Include pricing or detailed offers
- Use fonts smaller than 4" for key info
- Make it look cluttered or busy

## Typography Guidelines

**PRIMARY HEADLINE FONT:**
- Recommended: Montserrat Bold or similar clean sans-serif
- Fallback: Arial Bold, Helvetica Bold

**BODY COPY FONT:**
- Recommended: Open Sans or Source Sans Pro
- Fallback: Arial, Helvetica

**MINIMUM FONT SIZES:**
- Headlines: 24pt+ (varies by ad size)
- Subheadlines: 14-18pt
- Body copy: 10-12pt
- Fine print: 8pt minimum

## Image Guidelines

**POOL PHOTOGRAPHY REQUIREMENTS:**
- Resolution: 300 dpi minimum at print size
- Format: TIFF or high-quality JPEG
- Color space: CMYK for print
- Style: Professional, well-lit, aspirational
- Content: Crystal-clear water, clean lines, beautiful settings

**STOCK PHOTO SOURCES:**
- iStock/Getty Images
- Shutterstock
- Adobe Stock
- Manufacturer-provided images (with permission)

**CUSTOM PHOTOGRAPHY (Recommended):**
- Hire local photographer for completed projects
- Budget: $300-500 per shoot
- Capture: Multiple angles, different times of day, lifestyle shots

## Quick Reference: Print Advertising Checklist

### Before Launching Any Print Campaign:

- [ ] Verify brand colors match specifications (CMYK)
- [ ] Confirm tracking URL/phone is set up and working
- [ ] Proofread all copy (have 2 people review)
- [ ] Verify phone number and website are correct
- [ ] Check image resolution (300 dpi for print)
- [ ] Request proof from printer before full run
- [ ] Set up lead tracking spreadsheet/CRM tags
- [ ] Brief team on expected incoming leads
- [ ] Schedule follow-up cadence for campaign leads

### After Campaign Runs:

- [ ] Track all leads by source
- [ ] Calculate cost per lead
- [ ] Follow up with all leads within 24-48 hours
- [ ] Request feedback on how leads found you
- [ ] Evaluate ROI after 60-90 days
- [ ] Adjust future campaigns based on results
