### Performance Improvement Process

#### Performance Improvement Plan (PIP)

**Triggered by:**
- Quarterly score below 3.5
- Single project score below 3.0
- Safety incident
- Multiple complaints

**PIP Components:**

1. **Meeting with PM and Operations Manager**
   - Review deficiencies
   - Discuss root causes
   - Set improvement expectations

2. **Written Improvement Plan**
   - Specific issues identified
   - Required improvements
   - Timeline for improvement
   - Consequences of non-improvement

3. **Monitoring Period**
   - Duration: 60-90 days
   - Increased oversight
   - Regular check-ins
   - Progress documentation

4. **Resolution**
   - Improvement achieved: Return to standard status
   - No improvement: Removal from approved list

#### Recognition Program

**Preferred Subcontractor Benefits:**
- Priority scheduling
- Increased work allocation
- Rate consideration
- Annual recognition
- Reference letters

**Recognition Criteria:**
- Annual score > 4.5
- Zero safety incidents
- Zero major deficiencies
- Positive customer feedback
- Reliable partnership

## Probationary Period Process

### Probationary Period Overview

#### Purpose
- Evaluate new subcontractors in real project conditions
- Verify capabilities match qualifications
- Ensure cultural fit with TSAS standards
- Minimize risk on critical projects

#### Duration
- Standard: 3 projects or 90 days (whichever is longer)
- Extended: Up to 6 projects if concerns exist
- Early release: Exceptional performance may shorten

### Probationary Requirements

#### Project Assignment Guidelines

| Project Characteristic | Probationary Assignment |
|------------------------|------------------------|
| Complexity | Simple to moderate |
| Value | <$25,000 initially |
| Timeline | Non-critical path |
| Customer visibility | Lower profile |
| PM oversight | High availability |

#### Monitoring Requirements

| Activity | Frequency | Responsibility |
|----------|-----------|----------------|
| Site visits | Daily initially | PM |
| Quality checks | Each phase | PM |
| Progress reports | Daily | Subcontractor |
| Photo documentation | Extensive | Subcontractor |
| PM evaluation | After each project | PM |

### Probationary Evaluation Process

#### After Each Probationary Project

1. **PM Evaluation**
   - Complete project evaluation form
   - Score all metrics
   - Document any concerns
   - Recommend continuation or termination

2. **Operations Review**
   - Review PM evaluation
   - Check for patterns
   - Verify documentation
   - Approve recommendation

3. **Feedback Session**
   - Meet with subcontractor
   - Share performance feedback
   - Discuss any concerns
   - Set expectations for next project

**Minimum Requirements for Full Approval:**
- Average score > 3.5
- No safety incidents
- No major quality deficiencies
- Acceptable communication
- Positive PM recommendation

### Probationary Outcomes

#### Full Approval
**Criteria:**
- Meets minimum requirements
- PM recommends approval
- Operations concurs

**Process:**
1. PM submits approval recommendation
2. Operations Manager reviews and approves
3. Subcontractor notified
4. Status updated to "Approved"
5. Work restrictions lifted

#### Extended Probation
**Criteria:**
- Mixed performance
- Some concerns remain
- Potential for improvement

**Process:**
1. PM recommends extension
2. Meet with subcontractor
3. Define specific improvement areas
4. Set extension timeline (1-3 projects)
5. Monitor closely

#### Probation Failure
**Criteria:**
- Score below 3.0
- Safety incident
- Major quality failure
- Unacceptable communication
- PM recommends removal

**Process:**
1. PM documents recommendation
2. Operations Manager reviews
3. Final evaluation meeting
4. Written notification
5. Remove from approved list

### Appeals Process

#### Probation Decision Appeal

**Process:**
1. Subcontractor requests appeal in writing within 10 days
2. Appeal reviewed by Operations Manager
3. Meeting scheduled within 14 days
4. Subcontractor presents case
5. Final decision within 7 days

**Appeal Criteria:**
- New information not previously considered
- Procedural error in evaluation
- Extenuating circumstances

**Appeal Decision:**
- Upheld: Original decision stands
- Modified: Adjusted outcome
- Reversed: Original decision overturned
