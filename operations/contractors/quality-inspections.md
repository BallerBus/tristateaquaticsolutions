### Deficiency Communication Process

#### Notification Steps

1. **Identify Deficiency**
   - Document with photos
   - Complete deficiency report
   - Determine category

2. **Notify Subcontractor**
   - Immediate: Critical issues
   - Same day: Major issues
   - Within 24 hours: Minor issues

3. **Written Documentation**
   - Email deficiency report
   - Copy project file
   - Track in deficiency log

4. **Follow Up**
   - Verify correction made
   - Re-inspect
   - Close deficiency or escalate

### Repeated Deficiency Process

#### Pattern Recognition

When subcontractor has repeated deficiencies:

| Occurrence | Action |
|------------|--------|
| 2nd similar deficiency | Verbal warning, document |
| 3rd similar deficiency | Written warning, improvement plan required |
| 4th similar deficiency | Probation status |
| Continued pattern | Removal from approved list |

#### Performance Improvement Plan (PIP)

When PIP is triggered:

1. Meeting with subcontractor and PM
2. Review deficiency pattern
3. Identify root cause
4. Develop improvement steps
5. Set monitoring period (60-90 days)
6. Regular check-ins
7. Final determination
