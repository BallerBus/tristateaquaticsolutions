## Excavation Contractor Requirements

### 1. Qualifications

#### Required Credentials
- Valid state contractor's license with excavation classification
- Minimum 5 years experience in pool/spa excavation
- OSHA 30-hour certification for supervisors
- Competent Person certification for excavation safety

#### Equipment Requirements
| Equipment | Specification | Purpose |
|-----------|--------------|---------|
| Excavator | Mini to mid-size (varies by access) | Pool excavation |
| Skid steer | With various attachments | Finish grading, material handling |
| Dump trucks | Adequate capacity | Spoils removal |
| Compactor | Plate or roller | Backfill compaction |
| Laser level | Rotary type | Grade verification |

### 2. Pre-Work Requirements

#### Site Assessment
- [ ] Verify site access for equipment
- [ ] Confirm staging area location
- [ ] Identify utility locations
- [ ] Review soil boring reports (if available)
- [ ] Assess dewatering needs
- [ ] Determine spoils disposal method

#### Utility Coordination
1. **Call 811** - Minimum 3 business days before dig
2. **Private Locates** - Contract for private utility locating
3. **Mark Verification** - Verify all utilities marked on site
4. **Documentation** - Photograph all utility marks
5. **Hand Dig** - Within 24" of marked utilities

#### Documentation Review
- [ ] Approved pool/spa plans
- [ ] Survey showing property lines
- [ ] Setback requirements
- [ ] Depth specifications
- [ ] Slope requirements
- [ ] Soil type expectations

### 3. Excavation Procedures

#### Step 1: Layout and Staking
1. Establish baseline from house or property line
2. Stake pool corners and shape
3. Mark depth transitions
4. Verify setbacks from property lines
5. Photo document layout
6. Obtain PM approval before digging

#### Step 2: Equipment Staging
1. Position equipment for efficient operation
2. Protect existing surfaces (plywood on driveways)
3. Set up spoils removal route
4. Establish traffic control if needed
5. Place erosion control measures

#### Step 3: Excavation Sequence
1. **Strip topsoil** - Stockpile separately for reuse
2. **Rough excavation** - Dig to within 6" of final grade
3. **Shape formation** - Create walls, benches, slopes
4. **Fine grade** - Achieve final dimensions
5. **Inspection** - TSAS PM verification

#### Step 4: Dimensional Control
| Feature | Tolerance | Verification Method |
|---------|-----------|---------------------|
| Length/Width | +/- 2" | Tape measure from stakes |
| Depth | +/- 2" | Laser level from benchmark |
| Wall slope | Per spec +/- 2% | Slope gauge |
| Floor slope | Per spec +/- 1% | Laser level |
| Bench dimensions | +/- 1" | Direct measurement |

#### Step 5: Trench Excavation
1. **Plumbing trenches** - Per plumbing layout
2. **Electrical trenches** - Per electrical layout
3. **Equipment pad area** - Level and square
4. **Maintain slope** - For drainage

### 4. Dewatering Requirements

#### When Required
- High water table conditions
- During rainy season
- Soil types that hold water

#### Dewatering Methods
| Method | Application | Equipment |
|--------|-------------|-----------|
| Sump pump | Minor water intrusion | Submersible pump |
| Wellpoint | High water table | Wellpoint system |
| Deep well | Severe conditions | Deep well pump |

#### Discharge Requirements
- Identify legal discharge point
- Avoid neighbor's property
- Comply with stormwater regulations
- Document dewatering activities

### 5. Rock and Obstruction Handling

#### Rock Excavation
1. **Identification** - Document rock encounter
2. **Notification** - Notify PM immediately
3. **Documentation** - Photo and measure extent
4. **Method determination:**
   - Mechanical breaking (preferred)
   - Hydraulic hammer
   - Controlled blasting (rare, requires specialist)
5. **Pricing** - T&M or unit price per contract

#### Obstruction Protocol
1. Stop work immediately
2. Notify PM
3. Document obstruction
4. Assess removal method
5. Obtain approval before proceeding
6. Additional charges per contract terms

### 6. Safety Requirements

#### Excavation-Specific Safety
- **Competent Person** on site at all times
- **Sloping/Shoring** per soil classification
- **Access ladders** within 25' of workers
- **Spoils placement** 2' minimum from edge
- **Barricades** around open excavation
- **Daily inspections** documented

#### Equipment Safety
- Daily equipment inspection
- Only trained operators
- Maintain safe distances
- Use spotters for backing
- Secure equipment after hours

### 7. Completion Requirements

#### Final Inspection Checklist
- [ ] Dimensions within tolerance
- [ ] Walls stable and properly sloped
- [ ] Trenches at correct depth/location
- [ ] Site drainage adequate
- [ ] Area clean of debris
- [ ] Spoils removed or placed as specified

#### Documentation Package
- As-built dimensions
- Progress photos
- Utility locate records
- Dewatering logs (if applicable)
- Rock documentation (if applicable)
- Daily work logs

## Electrical Contractor Requirements

### 1. Qualifications

#### Required Credentials
- Valid state electrical contractor's license
- Experience with pool/spa electrical systems
- Familiarity with NEC Article 680 (Swimming Pools)
- OSHA 10-hour certification minimum

#### Insurance Requirements
- General liability: $1,000,000 minimum
- Umbrella: $2,000,000 (due to higher risk)
- Workers' compensation: Statutory
- Professional liability: $1,000,000 (if design included)

### 2. Code Compliance

#### NEC Article 680 Key Requirements

**Bonding Requirements (680.26)**
| Component | Requirement |
|-----------|-------------|
| Pool shell | Bonding grid with #8 AWG min |
| Metal parts | Bond all within 5' of pool |
| Water | Intentional bonding connection |
| Equipment | All metal equipment bonded |
| Deck | If conductive, bond reinforcing steel |

**GFCI Requirements (680.21, 680.22)**
| Equipment | Protection |
|-----------|------------|
| Pool pump motor | GFCI breaker required |
| Underwater lights | GFCI or isolation transformer |
| Outlets within 20' | GFCI protected |
| Spa equipment | GFCI breaker required |

**Clearances (680.8, 680.9)**
| Feature | Clearance |
|---------|-----------|
| Overhead conductors | 22.5' above water |
| Service equipment | 5' minimum from pool |
| Receptacles | 6' minimum from water |
| Light switches | 5' minimum from pool |

### 3. Pre-Work Requirements

#### Documentation Review
- [ ] Approved electrical plans
- [ ] Equipment specifications
- [ ] Panel/service capacity
- [ ] Permit obtained
- [ ] Utility coordination complete

#### Site Coordination
- [ ] Trench locations marked
- [ ] Equipment pad location confirmed
- [ ] Panel location confirmed
- [ ] Coordination with plumber on bonding

### 4. Installation Procedures

#### Step 1: Service/Panel Installation
1. **Assess existing service** - Capacity for pool loads
2. **Install sub-panel** - Per NEC requirements
3. **Size correctly:**
   | Equipment | Typical Load |
   |-----------|-------------|
   | Pool pump 1.5 HP | 240V, 20A |
   | Pool pump 2.5 HP | 240V, 30A |
   | Heater (gas) | 120V, 5A |
   | Heater (electric) | 240V, 60A+ |
   | Salt system | 240V, 20A |
   | Lights | 12V/120V, 15A |
4. **Label all circuits** - Clear identification
5. **Weatherproof** - NEMA 3R enclosure minimum

#### Step 2: Conduit Installation
1. **Size conduit** properly for wire fill
2. **Burial depth:**
   | Conduit Type | Minimum Depth |
   |--------------|---------------|
   | PVC Schedule 80 | 18" |
   | Rigid metal | 6" |
   | Direct burial cable | 18" |
3. **Use sweep fittings** - No sharp bends
4. **Protect above ground** - Schedule 80 or rigid
5. **Seal penetrations** - Weatherproof

#### Step 3: Bonding System
1. **Install bonding grid:**
   - #8 AWG solid copper minimum
   - Bare or green insulated
   - Connected to pool shell steel
2. **Bond all components:**
   - Rebar in pool shell
   - All metal within 5' of pool
   - Equipment housings
   - Deck reinforcing (if applicable)
3. **Water bonding:**
   - Install water bond fitting
   - Connect to bonding grid
4. **Test and verify:**
   - Continuity testing
   - Document all connections

#### Step 4: Equipment Circuits
1. **Pull wire** through conduit
2. **Make connections:**
   - Equipment terminals
   - Panel breakers
   - Use proper torque
3. **Install disconnects:**
   - Within sight of equipment
   - Minimum 5' from pool
4. **Label everything**

#### Step 5: Lighting Installation
1. **Niche installation:**
   - Coordinate with shell contractor
   - Proper bonding
   - Grounding
2. **Wire routing:**
   - Extra cord length for servicing
   - Proper forming
3. **Connection:**
   - Weatherproof junction box
   - GFCI protection or isolation transformer

### 5. Testing Requirements

#### Required Tests
| Test | Requirement | Documentation |
|------|-------------|---------------|
| Continuity | All bonding connections | Log all readings |
| Insulation resistance | All circuits | Megger test |
| GFCI function | All GFCI devices | Trip test |
| Polarity | All receptacles | Verify correct |
| Voltage | All circuits | Verify correct |

#### Documentation
- Test results log
- As-built drawings
- Equipment specs
- Manufacturer cut sheets
- Inspection cards

### 6. Inspection Requirements

#### Inspections Required
1. **Underground/rough-in** - Before backfill
2. **Bonding** - Before deck pour
3. **Final** - Before pool fill

#### Inspection Preparation
- All work accessible
- Test equipment on site
- Documentation ready
- Code book available

### 7. Safety Requirements

#### Electrical-Specific Safety
- **De-energize** before working
- **Lockout/Tagout** procedures
- **Test before touch** - Verify dead
- **Arc flash PPE** when required
- **Qualified workers** only

## Plumbing Contractor Requirements

### 1. Qualifications

#### Required Credentials
- Valid state plumbing contractor's license
- Pool/spa plumbing experience (5+ years)
- Familiarity with pool plumbing codes
- Gas license (if gas work included)

#### Skill Requirements
- PVC/CPVC welding
- Pressure testing
- Gas line installation (if applicable)
- Equipment connections
- Spa jet installation

### 2. Code Compliance

#### Key Code Requirements
| Requirement | Standard | Notes |
|-------------|----------|-------|
| VGB compliance | VGBA 2007/2008 | Drain covers, suction fittings |
| Backflow prevention | Local code | Fill line protection |
| Gas line | NFPA 54, local code | If applicable |
| Pressure testing | Local requirement | Document results |

#### VGB Act Compliance
1. **Drain covers** - ASME/ANSI A112.19.8 compliant
2. **Dual drains** - Required for single pump systems
3. **SVRS** - Safety vacuum release system (or dual drains)
4. **Documentation** - Certification of compliance

### 3. Pre-Work Requirements

#### Documentation Review
- [ ] Approved plumbing plans
- [ ] Equipment specifications
- [ ] Pipe sizing calculations
- [ ] VGB compliance plan
- [ ] Permit obtained

#### Site Coordination
- [ ] Trenches excavated
- [ ] Equipment pad location confirmed
- [ ] Shell penetration locations marked
- [ ] Bonding coordination with electrical

### 4. Installation Procedures

#### Step 1: Layout Verification
1. Review plans on site
2. Confirm equipment locations
3. Verify penetration locations
4. Mark pipe routing
5. Coordinate with steel/shell contractor

#### Step 2: Pool Shell Plumbing
**Main Drain:**
1. Install per VGB requirements
2. Dual drains with equalizer (if single pump)
3. Use compliant covers
4. Minimum pipe size: 2"
5. Slope toward equipment

**Skimmers:**
1. Position at prevailing wind end
2. Float valve equalizer line
3. Proper height for water level
4. Minimum pipe size: 2"

**Return Lines:**
1. Distribute evenly
2. Adjustable eyeball fittings
3. Size per flow requirements
4. Consider circulation patterns

**Spa Jets (if applicable):**
1. Position per plan
2. Correct jet type per specification
3. Air bleed manifold
4. Proper pipe sizing

#### Step 3: Equipment Pad Plumbing
1. **Union connections** at all equipment
2. **Valve manifold:**
   - Suction valves (skimmer, main drain, spa)
   - Return valves (pool, spa, water features)
3. **Equipment sequence:**
   - Pump suction
   - Pump discharge to filter
   - Filter to heater
   - Heater to chemical feeder
   - Return to pool/spa
4. **Bypass loops:**
   - Heater bypass
   - Solar bypass (if applicable)

#### Step 4: Fill Line
1. **Connection point** per local code
2. **Backflow preventer:**
   - Per local requirements
   - Testable device preferred
3. **Automatic fill valve:**
   - Install in accessible location
   - Adjustable water level
4. **Documentation** of backflow installation

#### Step 5: Gas Line (If Applicable)
1. **Size correctly** for BTU load
2. **Material:** Black iron, CSST, or approved
3. **Pressure test:** Per code requirement
4. **Drip leg** at equipment
5. **Shut-off valve** accessible
6. **Bonding** of CSST if required

### 5. Pressure Testing

#### Testing Requirements
| Test | Pressure | Duration | Standard |
|------|----------|----------|----------|
| Water pressure | 40 PSI | 24 hours | No loss |
| Air pressure | 10 PSI | 15 minutes | No loss |
| Gas pressure | Per code | Per code | Per code |

#### Testing Procedure
1. Cap all openings
2. Fill system with water (preferred) or air
3. Pressurize to test pressure
4. Monitor for specified duration
5. Document results
6. Repair any leaks
7. Re-test until passing

#### Documentation
- Date and time of test
- Initial and final pressure
- Ambient temperature
- Witness signatures
- Photos of pressure gauge

### 6. Inspection Requirements

#### Inspections Required
1. **Underground** - Before backfill
2. **Pressure test** - Witnessed
3. **Gas line** - Separate inspection
4. **Final** - Complete system

#### Inspection Preparation
- System pressurized
- All work visible/accessible
- Documentation ready
- Compliant materials

### 7. Quality Standards

#### Workmanship Requirements
| Item | Standard |
|------|----------|
| PVC joints | Fully welded, no dry fits |
| Pipe support | Per code, no sagging |
| Penetrations | Sealed, flashed |
| Valves | Accessible, labeled |
| Equipment connections | Union connections, no stress |

#### Flow Requirements
- Design flow rate achieved
- No air in system
- Proper circulation
- Balanced returns

## Concrete/Decking Contractor Requirements

### 1. Qualifications

#### Required Credentials
- Valid state contractor's license
- Concrete/flatwork experience (5+ years)
- Pool deck specific experience preferred
- OSHA 10-hour certification

#### Skill Requirements
- Forming and finishing
- Decorative concrete (stamped, colored)
- Coping installation
- Expansion/control joint design
- Drainage design

### 2. Pre-Work Requirements

#### Documentation Review
- [ ] Approved deck plans
- [ ] Finish specifications
- [ ] Grade and drainage plan
- [ ] Expansion joint layout
- [ ] Color/pattern selections

#### Site Coordination
- [ ] Pool shell complete
- [ ] Bond beam ready
- [ ] Electrical conduits in place
- [ ] Plumbing tested and approved
- [ ] Sub-grade prepared

### 3. Materials Specifications

#### Concrete Mix
| Property | Specification |
|----------|--------------|
| Compressive strength | 4,000 PSI minimum at 28 days |
| Slump | 4" maximum |
| Air entrainment | 4-6% (in freeze areas) |
| Cement content | 6 sack minimum |
| Water/cement ratio | 0.45 maximum |

#### Reinforcement Options
| Type | Application |
|------|-------------|
| Fiber mesh | Standard reinforcement |
| Wire mesh | 6x6 W2.9xW2.9 |
| Rebar | #4 @ 18" O.C. each way (heavy loads) |

### 4. Installation Procedures

#### Step 1: Sub-Grade Preparation
1. **Verify compaction** - 95% minimum
2. **Grade for drainage** - Minimum 1/4" per foot away from pool
3. **Install gravel base:**
   - 4" minimum compacted
   - Well-graded material
4. **Place vapor barrier** (if specified)
5. **Set forms:**
   - Straight and true
   - Proper height
   - Secure stakes

#### Step 2: Coping Installation
1. **Set coping on bond beam:**
   - Level and straight
   - Proper overhang (1-1.5")
   - Consistent reveal
2. **Mortar bed:**
   - Even thickness
   - Full bearing
3. **Joint between pieces:**
   - Consistent width
   - Prepared for grouting/sealing
4. **Expansion joint:**
   - Between coping and deck

#### Step 3: Concrete Placement

**Standard Broom Finish:**
1. Place concrete in sections
2. Strike off to grade
3. Bull float for initial finish
4. Edge all perimeters and joints
5. Broom finish for texture
6. Cut control joints
7. Apply curing compound

**Stamped Concrete:**
1. Place concrete
2. Apply color hardener (if integral color, mix with concrete)
3. Bull float
4. Apply release agent
5. Stamp pattern while plastic
6. Detail joints and edges
7. Wash off excess release
8. Apply sealer

**Decorative Options:**
| Finish | Process | Maintenance |
|--------|---------|-------------|
| Broom | Broom texture | Seal every 2-3 years |
| Stamped | Pattern stamps + color | Reseal annually |
| Exposed aggregate | Wash and expose | Seal every 2-3 years |
| Colored | Integral or broadcast | Seal per type |

#### Step 4: Control/Expansion Joints
1. **Control joints:**
   - Maximum 10' spacing
   - 1/4 depth of slab
   - Saw cut or tooled
2. **Expansion joints:**
   - At all fixed objects
   - Around pool edge
   - Pre-formed material

#### Step 5: Equipment Pad
1. **Form and pour separately** (recommended)
2. **Specifications:**
   - 4" minimum thickness
   - 4,000 PSI concrete
   - Level surface
   - Penetrations for conduit/plumbing
3. **Anchor bolts** if required

### 5. Curing Requirements

#### Curing Methods
| Method | Duration | Notes |
|--------|----------|-------|
| Curing compound | N/A | Apply immediately |
| Wet cure | 7 days | Continuous moisture |
| Plastic sheeting | 7 days | Keep moist underneath |

#### Protection During Cure
- No foot traffic: 24-48 hours minimum
- Light traffic: 3-7 days
- Heavy traffic: 7+ days
- Full cure: 28 days

### 6. Quality Standards

#### Dimensional Tolerances
| Item | Tolerance |
|------|-----------|
| Surface flatness | 1/4" in 10' |
| Slope | Per plan +/- 1/8" per foot |
| Thickness | +1/2", -1/4" |
| Joint straightness | 1/4" in 10' |

#### Visual Standards
- Uniform color (if colored)
- No cold joints
- No visible cracks
- Proper joint tooling
- Clean edges

### 7. Common Issues and Prevention

| Issue | Prevention |
|-------|------------|
| Cracking | Proper curing, control joints |
| Scaling | Proper finishing, no overworking |
| Discoloration | Consistent mix, curing |
| Delamination | Proper sub-grade prep |
| Settlement | Adequate compaction |

## Fencing Contractor Requirements

### 1. Qualifications

#### Required Credentials
- Valid contractor's license (if required by state)
- Fence installation experience (3+ years)
- Pool barrier code knowledge
- OSHA 10-hour certification

#### Knowledge Requirements
- Local pool barrier codes
- Property line determination
- Gate hardware requirements
- Material specifications

### 2. Code Compliance

#### Pool Barrier Requirements
| Requirement | Standard | Notes |
|-------------|----------|-------|
| Height | 48" minimum | From grade to top |
| Gap below fence | 4" maximum | To prevent crawling |
| Picket spacing | 4" maximum | No footholds |
| Non-climbable | Yes | No horizontal rails on outside |

#### Gate Requirements
| Requirement | Standard |
|-------------|----------|
| Self-closing | Required |
| Self-latching | Required |
| Latch height | 54" minimum from grade |
| Opening direction | Away from pool |
| Lock | Required (if property line gate) |

### 3. Pre-Work Requirements

#### Documentation Review
- [ ] Approved fence plan
- [ ] Survey showing property lines
- [ ] Setback requirements
- [ ] HOA restrictions (if applicable)
- [ ] Permit obtained

#### Site Coordination
- [ ] Property lines marked
- [ ] Utility locates complete
- [ ] Existing fence removal (if needed)
- [ ] Gate swing clearance verified

### 4. Installation Procedures

#### Step 1: Layout
1. **Verify property lines** with survey
2. **Mark post locations:**
   - Start at gate locations
   - Maximum 8' on center (6' preferred for pool fence)
   - Mark all corner posts
3. **Check clearances:**
   - Setback from property line
   - Clearance for gate swing
4. **Obtain approval** before digging

#### Step 2: Post Installation

**Setting Posts:**
1. **Dig holes:**
   - Diameter: 3x post width minimum
   - Depth: 1/3 of total post height minimum (or frost line)
2. **Set posts:**
   - Plumb in both directions
   - Proper depth
3. **Concrete setting:**
   - Crown top to shed water
   - Allow to cure 24-48 hours
4. **Alternative methods:**
   - Post anchors in existing concrete
   - Driven posts in soft soil

**Post Specifications:**
| Material | Post Size | Hole Depth |
|----------|-----------|------------|
| Aluminum | 2" x 2" | Per manufacturer |
| Vinyl | 5" x 5" | 30" minimum |
| Wood | 4" x 4" | 24-30" |
| Steel | 2-3/8" | Per manufacturer |

#### Step 3: Rail and Picket Installation

**Horizontal Rails:**
1. Install per manufacturer specifications
2. Level and straight
3. Secure fastening

**Pickets/Panels:**
1. Install with consistent spacing
2. Maximum 4" gap
3. Secure at top and bottom
4. Level tops (or follow grade)

#### Step 4: Gate Installation

**Gate Hanging:**
1. Install hinges on post
2. Hang gate level and plumb
3. Check swing clearance
4. Adjust for smooth operation

**Hardware Installation:**
1. **Self-closing hinges:**
   - Adjust tension
   - Test closing
2. **Self-latching mechanism:**
   - Install at 54" minimum
   - Test latching from both sides
3. **Lock (if required):**
   - Key-alike with other gates

#### Step 5: Final Adjustments
1. Check all post plumb
2. Verify all picket spacing
3. Test all gates
4. Touch up any scratches
5. Clean up site

|---------|--------------|
| Alloy | 6061-T6 or equivalent |
| Finish | Powder coated |
| Rail | 1" x 2" minimum |

### 6. Inspection Requirements

#### Pool Barrier Inspection

#### Documentation

### 7. Quality Standards

| Item | Standard |

### 1. Qualifications

#### Required Credentials
- Valid landscape contractor's license (if required)
- Irrigation license (if applicable)
- Experience with pool environment landscaping
- Knowledge of drought-tolerant plants (regional)

#### Skill Requirements
- Hardscape installation
- Irrigation system design/installation

- Drainage solutions
- Lighting integration

### 2. Pre-Work Requirements

#### Documentation Review
- [ ] Approved landscape plan
- [ ] Irrigation design
- [ ] Hardscape specifications

#### Site Coordination
- [ ] Pool and deck complete
- [ ] Fencing complete
- [ ] Final grade established
- [ ] Existing utilities marked

### 3. Plant Selection Guidelines

#### Plants to Avoid Near Pools
| Category | Examples | Reason |
|----------|----------|--------|
| Heavy shedding | Mulberry, cottonwood | Debris in pool |
| Invasive roots | Willow, bamboo, ficus | Damage to pool/deck |
| Thorny plants | Bougainvillea, roses | Safety hazard |
| Allergenic | Olive (fruiting) | Allergies |
| High maintenance | Many fruit trees | Debris, bees |

#### Recommended Pool-Area Plants
| Category | Examples | Benefits |
|----------|----------|----------|
| Palms | Date palm, fan palm | Tropical feel, low debris |
| Ornamental grasses | Fountain grass, blue fescue | Movement, low maintenance |
| Succulents | Agave, aloe | Drought tolerant, clean |
| Evergreen shrubs | Boxwood, ligustrum | Screening, year-round green |
| Groundcovers | Dymondia, creeping thyme | Deck edge softening |

### 4. Installation Procedures

#### Step 1: Site Preparation
1. **Clear debris** from work area
2. **Protect pool and deck:**
   - Cover pool
   - Protect deck surfaces
3. **Grade verification:**
   - Positive drainage away from pool
   - No low spots for standing water
4. **Soil preparation:**
   - Test soil
   - Amend as needed
   - Rototill planting areas

#### Step 2: Hardscape Installation

**Retaining Walls:**
1. Proper foundation
2. Drainage behind wall
3. Correct batter/setback
4. Capping/finishing

**Planters:**
1. Adequate drainage
2. Waterproofing against structures
3. Root barrier if needed

**Pathways:**
1. Proper base preparation
2. Edge restraints
3. Surface drainage
4. Non-slip finish

#### Step 3: Irrigation Installation

**System Components:**
| Component | Specification |
|-----------|--------------|
| Controller | Smart controller recommended |
| Valves | Separate zones for turf, shrubs, drip |
| Sprinkler heads | High efficiency, matched precipitation |
| Drip | 1-2 GPH emitters for shrubs |
| Rain sensor | Required in many areas |

**Installation Steps:**
1. Install mainline from water source
2. Install valve manifold
3. Run lateral lines
4. Install heads/emitters
5. Connect controller
6. Test and adjust
7. Program schedule

**Pool Area Considerations:**
- Direct spray away from pool
- Use drip near pool edge
- Prevent overspray onto deck
- Avoid runoff into pool

#### Step 4: Planting

**Tree Installation:**
1. Dig hole 2x root ball width
2. Set at proper depth (crown at grade)
3. Remove all wrapping
4. Backfill and water
5. Stake if needed (temporarily)
6. Mulch (not against trunk)

**Shrub Installation:**
1. Spacing per mature size
2. Root ball at or slightly above grade
3. Amend backfill if needed
4. Water thoroughly
5. Mulch 2-3" depth

**Groundcover/Perennials:**
1. Proper spacing
2. Root ball at grade
3. Water in well
4. Mulch between plants

#### Step 5: Mulch Application
1. **Depth:** 2-3" typical
2. **Type:** Bark, rock, or specified material
3. **Coverage:** All bare soil in beds
4. **Clearance:** Keep away from plant stems
5. **Edges:** Clean edge with deck/lawn

#### Step 6: Sod Installation (If Applicable)
1. **Soil prep:** Rake smooth, slight grade
2. **Lay sod:**
   - Stagger joints
   - Tight seams
   - Roll to ensure contact
3. **Water immediately:** Deeply
4. **Edge:** Along all borders
