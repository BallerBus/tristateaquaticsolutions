### Cash Flow Forecast Inputs

#### Inflow Sources

| Source | Forecast Method |
|--------|-----------------|
| Deposits | Signed contracts + 50% of proposals in negotiation |
| Progress Payments | Project schedule milestones |
| Final Payments | Project completion dates |
| Service Revenue | Historical average + scheduled work |
| Financing Draws | Per lender funding schedule |

#### Outflow Categories

| Category | Forecast Method |
|----------|-----------------|
| Payroll | Fixed based on headcount |
| Materials | Per project material schedules |
| Subcontractors | Per project schedules |
| Pool Shells | Manufacturer delivery/payment schedule |
| Fixed Overhead | Monthly recurring amounts |
| Variable Expenses | Historical averages + known items |

### Cash Management Guidelines

**Target Minimum Balance:** $50,000 (or 2 weeks operating expenses)

**When Cash is Tight:**
1. Accelerate receivables collection
2. Delay non-critical vendor payments (within terms)
3. Sequence pool shell orders carefully
4. Consider progress payment advances from customers
5. Draw on credit line if needed

**When Cash is Strong:**
1. Pay vendors early for discounts (if offered)
2. Build operating reserve
3. Consider equipment purchases
4. Pre-order materials if pricing favorable
5. Pay down credit line balance

### KPI Definitions and Targets

| KPI | Definition | Target | Alert Threshold |
|-----|------------|--------|-----------------|
| Gross Margin | (Revenue - Direct Costs) / Revenue | 45% | <35% |
| Net Margin | Net Profit / Revenue | 35% | <25% |
| AR Days | Average AR / (Annual Revenue / 365) | <30 days | >45 days |
| Labor % | Total Labor Cost / Revenue | <10% | >15% |
| Win Rate | Contracts Signed / Proposals Issued | >40% | <25% |
| Avg Contract Value | Total Contract Value / # Contracts | $80,000 | <$60,000 |

**Frequency:** Monthly

2. **Deep dive on problem areas** (20 min)
   - Why is KPI off target?
   - Root cause analysis
   - Action items

3. **Profitability review** (15 min)
   - Completed project margins
   - In-progress project status
   - Estimating accuracy

4. **Cash flow review** (10 min)
   - 13-week forecast
   - Any concerns
   - Upcoming large items

5. **Action items** (5 min)
   - Assign owners
   - Set deadlines

### Monthly Close Checklist

#### Pre-Close (Last Week of Month)

- [ ] Review all outstanding invoices
- [ ] Follow up on pending payments
- [ ] Review job costs for active projects
- [ ] Verify all payroll processed
- [ ] Ensure all vendor bills entered
- [ ] Review credit card statements

#### Close Process

##### Day 1: Cash and Banking
- [ ] Download all bank transactions
- [ ] Clear outstanding checks
- [ ] Research unidentified deposits
- [ ] Complete bank reconciliation
- [ ] Reconcile credit cards
- [ ] Record any bank fees/interest

##### Day 2: Receivables and Payables
- [ ] Run AR aging report
- [ ] Review and write off uncollectibles (if approved)
- [ ] Apply unapplied payments
- [ ] Run AP aging report
- [ ] Verify all bills entered
- [ ] Review vendor statements

##### Day 3: Job Costing
- [ ] Reconcile labor hours to payroll
- [ ] Verify material costs per job
- [ ] Allocate equipment costs
- [ ] Review subcontractor bills coded correctly
- [ ] Calculate WIP adjustments
- [ ] Review job profitability

##### Day 4: Accruals and Adjustments
- [ ] Accrue payroll (if pay period spans months)
- [ ] Accrue vacation/PTO liability
- [ ] Accrue known expenses not yet billed
- [ ] Depreciation entry
- [ ] Prepaid expense amortization
- [ ] Interest expense accrual
- [ ] Insurance expense allocation

##### Day 5: Review and Close
- [ ] Review trial balance
- [ ] Research unusual variances
- [ ] Verify balance sheet accounts
- [ ] Complete any adjusting entries
- [ ] Lock the period
- [ ] Create backup

#### Post-Close (Day 6-7)

##### Report Generation
- [ ] Income Statement (monthly and YTD)
- [ ] Balance Sheet
- [ ] Cash Flow Statement
- [ ] AR Aging Detail
- [ ] AP Aging Detail
- [ ] Job Profitability Summary
- [ ] KPI Dashboard

##### Management Review
- [ ] Compare actual to budget
- [ ] Analyze variances
- [ ] Identify trends
- [ ] Document issues and action items
- [ ] Present to management team
