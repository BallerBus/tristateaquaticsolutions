## Pre-Installation Checklist

### 7 Days Before Installation

- [ ] Confirm customer availability and site access
- [ ] Verify all permits are approved and on-site
- [ ] Confirm material delivery dates with vendors
- [ ] Review final design specifications with installation team
- [ ] Verify utility locate requests have been completed (call 811)
- [ ] Confirm equipment availability and schedule delivery
- [ ] Send customer pre-installation packet (timeline, expectations, contact info)

### 3 Days Before Installation

- [ ] Conduct site walkthrough (if not already done)
- [ ] Verify all materials have been received and inspected
- [ ] Confirm crew assignments and communicate schedule
- [ ] Check weather forecast and plan contingencies
- [ ] Verify access routes for equipment and materials
- [ ] Confirm portable restroom delivery (if required)
- [ ] Review HOA requirements (if applicable)

### 1 Day Before Installation

- [ ] Load materials and equipment on trucks
- [ ] Verify all tools and safety equipment are available
- [ ] Confirm start time with customer
- [ ] Brief crew on project specifics and safety considerations
- [ ] Verify all required documentation is prepared
- [ ] Charge batteries for tools and cameras
- [ ] Review traffic/parking plan with crew

### Day of Installation (Before Work Begins)

- [ ] Arrive 15 minutes early to assess conditions
- [ ] Conduct safety briefing with all crew members
- [ ] Walk perimeter with customer to document existing conditions
- [ ] Set up safety barriers and signage
- [ ] Verify utility locations match markings
- [ ] Establish material staging area
- [ ] Set up waste disposal containers
- [ ] Confirm customer emergency contact information

## Daily Setup Procedures

### Morning Arrival (All Days)

1. **Site Security Check** (5 minutes)
   - Inspect overnight security of materials and equipment
   - Check for vandalism or weather damage
   - Verify no unauthorized access occurred
   - Document any issues with photos

2. **Safety Briefing** (10 minutes)
   - Gather all crew members
   - Review day's specific tasks and hazards
   - Assign responsibilities and buddy system partners
   - Confirm understanding of emergency procedures
   - Distribute PPE as needed

3. **Equipment Inspection** (15 minutes)
   - Check all power tools for damage
   - Verify fuel levels in equipment
   - Inspect electrical cords for damage
   - Test safety features on all equipment
   - Document any equipment issues

4. **Work Area Preparation** (10 minutes)
   - Set up caution tape and barriers
   - Position equipment for day's work
   - Organize materials in work sequence
   - Ensure first aid kit and emergency equipment accessible

### Customer Communication (Daily)

- Brief check-in with homeowner at start of day
- Provide estimated work scope for the day
- Confirm any access needs or timing restrictions
- End-of-day summary of progress
- Photo documentation shared via customer preferred channel

## Phase-by-Phase Protocols

### Phase 1: Excavation

**Duration:** 1-3 days (varies by pool size and conditions)

**Procedures:**
1. Mark dig area according to approved plans
2. Verify all utility locations one final time
3. Set grade stakes and establish benchmarks
4. Begin excavation from center, working outward
5. Monitor soil conditions and document any variations
6. Set aside topsoil separately for later use
7. Check dimensions every 2 feet of depth
8. Install temporary safety fencing around excavation

**Quality Checkpoint:**
- Dimensions within ±2 inches of specifications
- Depth verified at minimum 6 points
- Photos of completed excavation from all angles
- Soil condition report completed

### Phase 2: Steel and Plumbing

**Duration:** 2-4 days

**Procedures:**
1. Install base drain(s) according to specifications
2. Lay out steel reinforcement per engineering drawings
3. Maintain proper clearances from shell edge
4. Tie all intersections with approved ties
5. Install main drain anti-entrapment cover
6. Pressure test all plumbing lines to 25 PSI
7. Install bonding wire and verify continuity
8. Document all plumbing runs with photos

**Quality Checkpoint:**
- Steel spacing verified at multiple points
- Plumbing pressure test holds for minimum 4 hours
- Bonding continuity tested and documented
- City/county inspection passed (if required)

### Phase 3: Gunite/Shotcrete Application

**Duration:** 1-2 days

**Procedures:**
1. Verify weather conditions meet requirements (no rain, >40°F)
2. Pre-wet excavation area
3. Apply shell material per thickness specifications
4. Maintain consistent application throughout
5. Hand finish all surfaces requiring tile or finish
6. Begin curing process immediately
7. Protect fresh shell from debris and damage

**Quality Checkpoint:**
- Thickness verified at specified intervals
- Surface consistency and smoothness verified
- No visible voids, cracks, or delamination
- Cure process documented and scheduled

### Phase 4: Tile and Coping

**Duration:** 2-4 days

**Procedures:**
1. Verify shell has cured minimum required time
2. Layout tile pattern according to design
3. Install waterline tile with proper adhesive
4. Install coping stones with appropriate mortar
5. Grout all tile and seal as required
6. Protect completed work from damage

**Quality Checkpoint:**
- Tile alignment verified level
- Grout lines consistent throughout
- Coping elevation consistent (±1/8")
- No loose tiles or coping stones

### Phase 5: Decking and Hardscape

**Duration:** 3-5 days

**Procedures:**
1. Prepare subgrade per specifications
2. Install control joints as designed
3. Pour concrete or install pavers per plan
4. Apply texture finish consistently
5. Cut expansion joints as required
6. Apply sealer per manufacturer requirements
7. Protect fresh concrete from traffic

**Quality Checkpoint:**
- Proper drainage slope verified (away from pool)
- Expansion joints at required intervals
- Texture consistent throughout
- No cracks or defects in finished surface

### Phase 6: Equipment and Startup

**Duration:** 1-2 days

**Procedures:**
1. Install all equipment per manufacturer specifications
2. Complete all electrical connections (by licensed electrician)
3. Fill pool with water to operating level
4. Prime and start pump system
5. Balance water chemistry per startup specifications
6. Program automation system per customer preferences
7. Verify all equipment operation
8. Complete final inspection checklist

**Quality Checkpoint:**
- All equipment operating within specifications
- No leaks in plumbing or equipment
- Water circulation verified throughout pool
- Automation system functioning correctly

## Safety Requirements

### Mandatory PPE

- Hard hats during excavation and overhead work
- Safety glasses at all times on site
- Steel-toed boots required
- High-visibility vests when traffic present
- Hearing protection near loud equipment
- Gloves appropriate to task
- Dust masks during gunite application

### Daily Safety Requirements

- Safety briefing before work begins
- First aid kit accessible on site
- Fire extinguisher accessible on site
- Emergency contact numbers posted
- Utility locations verified before digging
- Trench safety measures when depth exceeds 4 feet

### Equipment Safety

- Only trained operators on power equipment
- Lockout/tagout procedures for maintenance
- Daily equipment inspections
- Proper lifting techniques for materials
- No equipment operation near excavation edge
- Grounding verification for electrical equipment

See [Safety Manual](safety-manual.md) for complete safety procedures.

## Documentation Requirements

### Daily Documentation

- Progress photos (minimum 10 per day)
- Daily log with work completed
- Material usage tracking
- Equipment hours log
- Weather conditions recorded
- Any issues or deviations noted
- Customer communications logged

### Phase Documentation

- Phase completion photos
- Inspection reports and approvals
- Quality checkpoint sign-offs
- Material certifications collected
- Test results (pressure, electrical, etc.)
- Change order documentation (if applicable)

### Project Completion Documentation

- Complete photo documentation organized by phase
- As-built drawings (if variations from plan)
- All permit sign-offs and approvals
- Equipment manuals and warranty cards
- Customer training acknowledgment
- Final inspection sign-off
- Warranty documentation package

### Photo Requirements

- Minimum resolution: 4MP
- Include date/time stamp
- Include reference scale where appropriate
- Capture from multiple angles
- Label photos with phase and description
- Upload to project folder within 24 hours

## Clean-Up Procedures

### Daily Clean-Up

- [ ] Collect all debris and place in designated containers
- [ ] Store all tools and equipment securely
- [ ] Cover exposed materials from weather
- [ ] Clean work area of hazards
- [ ] Secure site perimeter fencing
- [ ] Clear customer access paths

### Phase Completion Clean-Up

- [ ] Remove all phase-specific debris
- [ ] Clean adjacent areas affected by work
- [ ] Remove temporary utilities no longer needed
- [ ] Grade and smooth disturbed soil
- [ ] Verify no materials left in pool area

### Final Clean-Up

- [ ] Remove all construction debris from site
- [ ] Clean all pool surfaces and surrounding areas
- [ ] Remove all staging materials and equipment
- [ ] Restore any damaged landscaping
- [ ] Clean pool deck and coping of construction residue
- [ ] Final power wash of all completed surfaces
- [ ] Remove all signage and temporary fencing
- [ ] Verify no materials or equipment left on site
- [ ] Obtain customer sign-off on site condition
