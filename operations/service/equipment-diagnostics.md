# Equipment Diagnostics

## Tri-State Aquatic Solutions - Troubleshooting Guide

## Pump Troubleshooting

### Pump Won't Start

**Symptom:** Motor makes no sound, no humming

| Check | Cause | Solution |
|-------|-------|----------|
| Power at outlet | No power | Check breaker, GFCI, timer |
| Timer/automation | Timer off | Adjust timer or override |
| Capacitor | Failed capacitor | Replace start capacitor |
| Motor windings | Burned motor | Replace motor |
| Switch/relay | Failed switch | Replace switch/relay |

**Diagnostic Steps:**
1. Verify power at outlet with multimeter
2. Check timer and automation settings
3. Listen for hum when power applied
4. Test capacitor with meter
5. Check motor continuity
6. Smell for burned windings

---

### Pump Hums But Won't Start

**Symptom:** Motor hums, may trip breaker

| Check | Cause | Solution |
|-------|-------|----------|
| Impeller | Debris jammed | Clear impeller obstruction |
| Shaft | Frozen shaft | Free shaft, replace seal/bearings |
| Capacitor | Weak capacitor | Replace capacitor |
| Voltage | Low voltage | Check electrical supply |
| Motor | Seized bearings | Replace motor |

**Diagnostic Steps:**
1. Turn off power completely
2. Remove pump lid and basket
3. Reach into volute to check impeller rotation
4. If impeller won't turn, check for debris
5. If clear but won't spin, shaft/bearings seized
6. Test capacitor

---

### Pump Loses Prime

**Symptom:** Pump basket not full, air bubbles in pump

| Check | Cause | Solution |
|-------|-------|----------|
| Water level | Low pool water | Fill pool to mid-skimmer |
| Pump lid | Bad lid O-ring | Replace/lubricate O-ring |
| Pump lid | Cracked lid | Replace lid |
| Suction leak | Air leak in suction line | Pressure test, repair leak |
| Skimmer weir | Stuck weir door | Free or remove weir |
| Valve position | Closed valve | Open suction valves |
| Clogged line | Blocked suction | Clear obstruction |

**Diagnostic Steps:**
1. Check water level first
2. Inspect lid O-ring for cracks/flat spots
3. Apply shaving cream around fittings (bubbles = leak)
4. Check all suction valves are open
5. Close drain, run from skimmer only
6. Pressure test suction line if needed

---

### Pump Making Noise

**Symptom:** Grinding, screeching, or cavitation sounds

| Noise Type | Cause | Solution |
|------------|-------|----------|
| Grinding | Bad bearings | Replace motor or bearings |
| Screeching | Dry bearings | Motor replacement |
| Cavitation (gravel sound) | Starved for water | Improve suction, check lines |
| Vibration | Loose base, bad mounts | Tighten, replace mounts |
| Clicking | Debris in impeller | Clear impeller |

**Diagnostic Steps:**
1. Identify noise type and location
2. Check for vibration at motor
3. Feel motor temperature
4. Verify full prime
5. Check impeller for debris
6. Assess bearing play

---

### Pump Repair vs. Replace Decision

**Repair When:**
- Pump is less than 5 years old
- Single point of failure
- Quality pump brand
- Repair cost < 50% of replacement
- Motor only issue on good wet end

**Replace When:**
- Pump is over 8 years old
- Multiple failure points
- Shaft seal AND motor issue
- Pump is oversized (energy waste)
- Variable speed upgrade makes sense
- Repair cost > 50% of replacement

## Filter Troubleshooting

### High Filter Pressure

**Normal Range:** 10-20 PSI (varies by system)
**High Pressure:** 10+ PSI above clean baseline

| Check | Cause | Solution |
|-------|-------|----------|
| Filter media | Dirty filter | Backwash, clean, or replace media |
| Return valves | Closed valve | Open return valves |
| Return lines | Obstruction | Clear blockage |
| Gauge | Faulty gauge | Replace pressure gauge |
| Filter grids/cartridge | Calcified media | Acid wash or replace |

**Diagnostic Steps:**
1. Note current pressure reading
2. Backwash (sand/DE) or clean cartridge
3. Check pressure after cleaning
4. If still high, check return side
5. If persistent, filter media needs replacement

---

### Low Filter Pressure

**Low Pressure:** 5+ PSI below normal baseline

| Check | Cause | Solution |
|-------|-------|----------|
| Pump basket | Clogged basket | Clean pump basket |
| Skimmer basket | Clogged basket | Clean skimmer basket |
| Impeller | Clogged impeller | Clear impeller |
| Suction leak | Air in system | Find and repair leak |
| Water level | Low water | Fill pool |
| Pump issue | Weak pump | Diagnose pump |

**Diagnostic Steps:**
1. Clean all baskets
2. Check water level
3. Look for air bubbles
4. Check pump prime
5. Clear impeller if needed
6. Verify pump operation

---

### Filter Returning Debris to Pool

**Symptom:** Dirt, algae, or DE visible in pool

| Filter Type | Cause | Solution |
|-------------|-------|----------|
| Sand | Broken laterals | Replace laterals |
| Sand | Channeling | Replace sand |
| DE | Torn grids | Replace grids |
| DE | Damaged manifold | Replace manifold |
| Cartridge | Torn cartridge | Replace cartridge |
| All types | Multiport failure | Replace multiport valve |

**Diagnostic Steps:**
1. Identify debris type (sand = lateral, DE = grid)
2. Check multiport for crossover
3. Inspect filter media
4. Check for manifold cracks
5. Replace damaged components

---

### Filter Type Maintenance Schedule

| Filter Type | Routine Service | Media Replacement |
|-------------|-----------------|-------------------|
| Sand | Backwash monthly | Every 5-7 years |
| DE | Backwash monthly, annual breakdown | Every 2-3 years (grids) |
| Cartridge | Clean monthly | Every 1-2 years |

## Heater Troubleshooting

### Heater Won't Ignite

**Symptom:** No flame, no heat

| Check | Cause | Solution |
|-------|-------|----------|
| Power | No power | Check breaker, switches |
| Gas supply | No gas | Check gas valve, supply |
| Igniter | Failed igniter | Replace igniter |
| Flame sensor | Dirty sensor | Clean flame sensor |
| Pressure switch | Low flow | Check flow, adjust PSI switch |
| Control board | Board failure | Replace control board |
| High limit | Tripped high limit | Reset, check for cause |

**Diagnostic Steps:**
1. Verify power and gas supply
2. Check for error codes on display
3. Listen for ignition attempt
4. Verify water flow through heater
5. Check pressure switch setting
6. Test igniter spark
7. Clean flame sensor
8. Check gas valve operation

---

### Heater Error Codes (Common)

**Pentair MasterTemp:**
- E01: Ignition failure
- E02: Ignition lockout (3 failed attempts)
- E03: Flame sense fault
- E04: High limit tripped
- E05: Stack flue sensor fault

**Hayward H-Series:**
- IF: Ignition failure
- HF: High limit fault
- SF: Flame sense fault
- HS: High limit switch open
- PS: Pressure switch fault

**Raypak:**
- IGN: Ignition failure
- OPEN: High limit open
- LLCO: Low limit cutoff
- FL: Flame failure

---

### Heater Cycling On/Off

**Symptom:** Heater starts then shuts off repeatedly

| Check | Cause | Solution |
|-------|-------|----------|
| Flow rate | Low flow | Clean filter, check pump |
| Pressure switch | Needs adjustment | Adjust pressure switch |
| High limit | Overheating | Check ventilation, scaling |
| Gas pressure | Incorrect pressure | Adjust gas regulator |
| Heat exchanger | Scaled/blocked | Clean or replace exchanger |
| Thermistor | Faulty sensor | Replace thermistor |

**Diagnostic Steps:**
1. Check flow rate and filter pressure
2. Verify pressure switch clicks on/off
3. Check for scale buildup
4. Measure gas pressure
5. Inspect heat exchanger
6. Test thermistor resistance

---

### Heat Pump Issues

**Won't Heat:**
1. Check thermostat setting
2. Verify air flow (no obstructions)
3. Check refrigerant charge
4. Test compressor operation
5. Verify defrost cycle
6. Check water flow

**Icing Up:**
- Normal in cold weather (defrost cycle)
- Abnormal: Low refrigerant, blocked airflow, fan motor issue

**Noisy Operation:**
- Fan blade issue
- Compressor failing
- Vibration isolation needed

## Salt System Troubleshooting

### Salt Cell Not Generating

**Symptom:** Low chlorine, no cell activity

| Check | Cause | Solution |
|-------|-------|----------|
| Salt level | Low salt | Add salt to 3,200-3,400 ppm |
| Cell condition | Scaled cell | Clean cell with acid |
| Cell age | Worn cell | Replace cell (3-5 year life) |
| Flow sensor | No flow detected | Check flow, clean sensor |
| Control board | Board issue | Replace board |
| Temperature | Water too cold | Wait for warmer temps (>60\u00b0F) |

**Diagnostic Steps:**
1. Test salt level with independent meter
2. Verify system salt reading
3. Inspect cell for scale buildup
4. Check flow indicator
5. Verify cell voltage/amperage
6. Test with known good cell

---

### Salt Level Issues

**High Salt Reading (but water tests low):**
- Contaminated sensor
- Dirty cell
- Control board calibration

**Low Salt Reading (but water tests correct):**
- Dirty flow sensor
- Temperature affecting reading
- Board calibration issue

**Actual Low Salt:**
- Splash out/backwash
- Heavy rain dilution
- Leak and fresh water added

---

### Salt Cell Cleaning Procedure

**When to Clean:**
- Every 3-4 months
- When cell shows scale
- When output drops
- Manufacturer recommendation

**Cleaning Steps:**
1. Turn off system
2. Remove cell from housing
3. Inspect for scale buildup
4. Mix acid solution (4:1 water to muriatic)
5. Soak cell 5-10 minutes
6. Rinse thoroughly
7. Reinstall cell
8. Restart system

**Warning:** Always add acid to water, never water to acid.

---

### Salt Cell Life Expectancy

| Condition | Expected Life |
|-----------|---------------|
| Well-maintained, balanced water | 5-7 years |
| Average maintenance | 3-5 years |
| Poor chemistry, no cleaning | 2-3 years |

## Automation Troubleshooting

### System Won't Communicate

**Symptom:** No connection to app/remote

| Check | Cause | Solution |
|-------|-------|----------|
| WiFi connection | Lost WiFi | Reconnect to network |
| Firmware | Outdated firmware | Update firmware |
| Router | Router reset | Reconfigure network |
| Antenna | Poor signal | Relocate or add antenna |
| Account | App issue | Re-login, reinstall app |
| Hardware | Module failure | Replace communication module |

**Diagnostic Steps:**
1. Check WiFi signal at equipment
2. Verify network credentials
3. Restart automation system
4. Check app and firmware versions
5. Try local control
6. Test with different device

---

### Equipment Won't Respond to Commands

**Symptom:** Commands sent, no action

| Check | Cause | Solution |
|-------|-------|----------|
| Power | No power to equipment | Check circuits |
| Relay | Failed relay | Replace relay |
| Wiring | Loose connection | Tighten connections |
| Address | Wrong device ID | Verify addressing |
| Interlock | Safety interlock active | Check settings |

**Diagnostic Steps:**
1. Try manual operation
2. Check automation display for errors
3. Verify equipment power
4. Test relay operation
5. Check wiring connections
6. Review interlock settings

## When to Repair vs. Replace

### General Guidelines

**Repair When:**
- Equipment is less than 1/2 of expected life
- Single component failure
- Repair cost < 50% of replacement
- Parts readily available
- No efficiency gain from replacement

**Replace When:**
- Equipment is past 3/4 of expected life
- Multiple components failing
- Repair cost > 50% of replacement
- Energy efficiency upgrade available
- Technology obsolete
- Parts discontinued

## Diagnostic Tools Required

### Basic Diagnostic Kit

- Digital multimeter
- Amp clamp meter
- Pressure gauge (0-60 PSI)
- Salt/TDS meter
- Temperature probe
- Leak detection dye
- Shaving cream (air leaks)

### Advanced Diagnostic Kit

- Manometer (gas pressure)
- Megohmmeter (insulation testing)
- Pressure test kit (plumbing)
- Listening device (leak detection)
- Thermal imaging camera
- Clamp-on flow meter
