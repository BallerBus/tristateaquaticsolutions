## Reorder Points and Procedures

### Reorder Point System

**Reorder Point (ROP) Formula:**
```
ROP = (Average Daily Usage x Lead Time) + Safety Stock
```

**Example:**
- Average daily usage: 0.5 units
- Lead time: 7 days
- Safety stock: 2 units
- ROP = (0.5 x 7) + 2 = 5.5 = 6 units

### Reorder Triggers

**Daily Check:**
- Review parts used from yesterday
- Update inventory counts
- Generate reorder alerts
- Place urgent orders

**Weekly Review:**
- Full inventory count (rotating)
- Trend analysis
- Adjust reorder points seasonally
- Review slow-moving stock

**Monthly Analysis:**
- Complete inventory audit
- Cost analysis
- Supplier performance review
- Dead stock identification

## Seasonal Inventory Adjustments

### Spring (March - May)

**Increase Stock:**
- Opening chemicals
- Salt (heavy usage)
- Pump seals (startup failures)
- Filter cartridges
- Winter cover hardware

**Decrease Stock:**
- Winterization chemicals
- Antifreeze

### Summer (June - August)

**Increase Stock:**
- Chlorine and acid
- Salt cells
- Pump motors
- All chemicals
- Repair parts (peak usage)

**Maintain:**
- All standard inventory

### Fall (September - November)

**Increase Stock:**
- Closing chemicals
- Winter cover supplies
- Winterization plugs
- Antifreeze

**Decrease Stock:**
- Opening supplies
- Reduce chemical inventory

### Winter (December - February)

**Reduce Stock:**
- Chemicals (minimal)
- Consumable parts
- Keep emergency minimums

**Maintain:**
- Equipment parts for off-season repairs
- Indoor pool supplies
