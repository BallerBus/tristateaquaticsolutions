## Vendor Evaluation Criteria

### Initial Vendor Qualification

Before adding a vendor to the approved list, evaluate:

#### Mandatory Requirements (Must Pass All)

| Requirement | Verification Method |
|-------------|---------------------|
| Valid business license | Copy on file |
| General liability insurance ($1M minimum) | COI on file |
| Workers compensation insurance | COI on file |
| No outstanding legal issues | Background check |
| Positive references (minimum 3) | Phone verification |
| Financial stability | Credit check |

#### Scored Criteria (Minimum Score: 70/100)

| Criteria | Weight | Scoring |
|----------|--------|---------|
| Product/Service Quality | 25% | 1-5 scale |
| Pricing Competitiveness | 20% | 1-5 scale |
| Delivery Reliability | 15% | 1-5 scale |
| Lead Times | 15% | 1-5 scale |
| Customer Service | 10% | 1-5 scale |
| Technical Support | 10% | 1-5 scale |
| Payment Terms | 5% | 1-5 scale |

**Score Calculation:**
Score = (Quality × 25 + Price × 20 + Delivery × 15 + Lead × 15 + Service × 10 + Support × 10 + Terms × 5) / 5

### Annual Vendor Review

All approved vendors undergo annual review:

#### Review Components

1. **Performance Metrics Review**
   - On-time delivery rate (target: 95%+)
   - Quality rejection rate (target: <2%)
   - Order accuracy rate (target: 98%+)
   - Response time to inquiries (target: <24 hours)

2. **Issue History Review**
   - Number of complaints logged
   - Resolution time average
   - Repeat issues
   - Escalations required

3. **Pricing Analysis**
   - Price changes over year
   - Comparison to market rates
   - Value assessment

4. **Relationship Assessment**
   - Communication quality
   - Flexibility and responsiveness
   - Partnership attitude
   - Innovation and improvement

## Order Procedures

### Standard Order Process

```
1. Need Identified
   - Project Manager submits material request
   - Or inventory reaches reorder point
        ↓
2. Order Preparation (Purchasing)
   - Verify specifications
   - Check preferred vendor availability
   - Confirm pricing (use current price list or quote)
   - Prepare purchase order
        ↓
3. Approval (if required)
   - Orders > $2,500 require manager approval
   - Orders > $10,000 require executive approval
        ↓
4. Order Placement
   - Submit via vendor's preferred method
   - Obtain order confirmation
   - Log in tracking system
        ↓
5. Order Tracking
   - Confirm ship date
   - Track delivery
   - Notify site of expected arrival
        ↓
6. Receipt & Inspection
   - Verify quantities
   - Inspect quality
   - Document any issues
   - Sign delivery receipt (noting exceptions)
        ↓
7. Issue Resolution (if needed)
   - Report discrepancies same day
   - Document with photos
   - Obtain credit/replacement
        ↓
8. Payment Processing
   - Match invoice to PO and receipt
   - Process per payment terms
```

### Purchase Order Requirements

All orders over $500 require a formal purchase order.

**PO Must Include:**
- PO number (system-generated)
- Vendor name and address
- Ship-to address
- Bill-to address
- Order date
- Required delivery date
- Line items with:
  - Description
  - Part/item number
  - Quantity
  - Unit price
  - Extended price
- Special instructions
- Total amount
- Approver signature (if required)
- Terms and conditions reference

### Rush Order Procedures

For orders needed sooner than standard lead time:

1. **Identify Rush Need**
   - Document reason for rush
   - Confirm date needed

2. **Contact Vendor Directly**
   - Call vendor rep personally
   - Explain situation and need
   - Confirm they can meet date
   - Get confirmation in writing

3. **Accept Additional Costs**
   - Expedite fees may apply
   - Document approved upcharges
   - Get cost approval if exceeds limits

4. **Place and Track**
   - Mark order as RUSH
   - Get ship confirmation immediately
   - Track shipment actively
   - Confirm delivery receipt

5. **Document**
   - Note rush reason in project file
   - Capture any cost impacts
   - Lessons learned if process failure caused rush

## Quality Requirements

### Incoming Inspection Standards

#### Visual Inspection (All Deliveries)

- [ ] Packaging intact, no damage
- [ ] Correct items received (match PO)
- [ ] Correct quantities received
- [ ] No visible defects or damage
- [ ] Proper labeling
- [ ] Documentation included

#### Material-Specific Inspections

**Steel/Rebar:**
- [ ] Correct gauge/size
- [ ] No rust or corrosion
- [ ] Straight, not bent
- [ ] Certification paperwork included

**Plumbing Materials:**
- [ ] Correct sizes
- [ ] No cracks or defects
- [ ] Proper ratings marked
- [ ] Fittings match pipes

**Tile/Coping:**
- [ ] Color match verified
- [ ] No chips or cracks
- [ ] Quantity per box verified
- [ ] All boxes from same lot

**Equipment:**
- [ ] Model number matches order
- [ ] Serial number recorded
- [ ] No shipping damage
- [ ] All components included
- [ ] Manuals and warranty cards present

**Fiberglass Shells:**
- [ ] Color matches selection
- [ ] No gelcoat cracks or damage
- [ ] Dimensions verified
- [ ] All components included
- [ ] Transportation damage documented

### Rejection Procedures

When materials fail inspection:

1. **Document Immediately**
   - Photos of defects
   - Note on delivery receipt
   - Driver signature acknowledging issue

2. **Segregate Materials**
   - Mark as "ON HOLD - DO NOT USE"
   - Store separately from approved materials

3. **Notify Vendor Same Day**
   - Phone call to rep
   - Written notification (email)
   - Include photos and documentation

4. **Obtain Resolution**
   - Credit memo for partial shipments
   - Replacement shipment date
   - Return authorization if needed

5. **Track to Closure**
   - Credit received
   - Replacement received
   - Issue logged in vendor file

### Quality Expectations by Category

| Category | Acceptable Defect Rate | Action Threshold |
|----------|----------------------|------------------|
| Pool Equipment | 0% | Any defect = replacement |
| Structural Materials | <1% | >1% = vendor review |
| Tile/Finish Materials | <2% | >2% = credit/replacement |
| Consumables | <3% | >3% = vendor discussion |
| General Supplies | <5% | >5% = evaluate vendor |

### Vendor Quality Issues

When quality issues exceed thresholds:

1. **First Occurrence**
   - Document and notify vendor
   - Request corrective action plan
   - Continue ordering with increased inspection

2. **Second Occurrence (within 6 months)**
   - Formal warning letter
   - Required corrective action meeting
   - Probationary status

3. **Third Occurrence**
   - Review for removal from approved list
   - Source alternative vendors
   - Phased reduction of orders

4. **Critical Issue (any occurrence)**
   - Immediate stop on new orders
   - Executive review
   - Potential removal from vendor list

## Payment Terms

### Standard Payment Terms

| Vendor Type | Standard Terms | Early Pay Discount |
|-------------|---------------|-------------------|
| Major Suppliers | Net 30 | 2% 10, Net 30 |
| Subcontractors | Net 30 | Case by case |
| Equipment Dealers | Net 30 or 50% deposit | Per agreement |
| Small Suppliers | Net 15-30 | Varies |

### Payment Schedule

| Day | Action |
|-----|--------|
| Receipt | Invoice logged into system |
| +3 days | Invoice matched to PO and receiving |
| +5 days | Discrepancies resolved |
| +7 days | Invoice approved for payment |
| +10 days | Early pay discount payments processed |
| +25 days | Net 30 payments queued |
| +30 days | Payment issued |

### Payment Methods

| Method | Use Case | Processing Time |
|--------|----------|-----------------|
| ACH/Direct Deposit | Preferred for all vendors | 1-2 business days |
| Check | When required by vendor | 3-5 business days |
| Credit Card | Emergency purchases only | Immediate |
| Wire Transfer | Large/international orders | Same day |

### Invoice Requirements

Invoices must include:
- Vendor name and address
- Invoice number and date
- Our PO number
- Itemized list matching PO
- Unit prices matching PO
- Correct extensions and total
- Tax where applicable
- Payment terms stated
- Remit-to address

**Invoices lacking required information will be returned for correction.**

### Dispute Resolution

For invoice discrepancies:

1. **Identify Discrepancy**
   - Price difference
   - Quantity difference
   - Item not received
   - Quality issue credit due

2. **Document**
   - Original PO
   - Receiving documentation
   - Invoice in question
   - Calculation of discrepancy

3. **Notify Vendor**
   - Contact AP rep
   - Provide documentation
   - Request credit memo

4. **Partial Payment Option**
   - Pay undisputed portion
   - Note disputed amount
   - Resolve within 30 days

5. **Escalation**
   - If not resolved in 30 days
   - Escalate to vendor management
   - Consider impact on future orders

### New Vendor Setup

Before first payment can be processed:

Required Documentation:
- [ ] W-9 form
- [ ] Vendor application completed
- [ ] Bank information for ACH (preferred)
- [ ] COI on file
- [ ] Credit application (if requesting terms)
- [ ] Approved vendor evaluation

Processing Time: 5-7 business days for new vendor setup

### Payment Authorization Levels

| Amount | Required Approval |
|--------|-------------------|
| < $2,500 | Purchasing Manager |
| $2,500 - $10,000 | Operations Manager |
| $10,000 - $25,000 | CFO/Controller |
| > $25,000 | Executive + CFO |
