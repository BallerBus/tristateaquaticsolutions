Conversion Components Guide

Reusable lead capture and conversion optimization components for the Tri-State Aquatic Solutions website. Each component is self-contained with HTML, CSS, and JavaScript.

Component Overview

Lead Capture

Exit Intent Popup

Triggers when user moves their mouse to leave the page. Offers a free cost estimate in exchange for contact info.

File: /components/exit-intent-popup.html

Behavior:

  • Detects mouse leaving viewport (desktop) or rapid scroll-up (mobile)
  • Shows once per 7 days (localStorage tracking)
  • Smooth fade/scale animation
  • Accessible with keyboard support
Urgency

Seasonal Banner

Top-of-page banner with countdown timer. Creates urgency around seasonal installation deadlines.

File: /components/seasonal-banner.html

Features:

  • Countdown to target date (March 15, 2026 for Memorial Day swim)
  • Dismissible (respects for 3 days)
  • Responsive design
  • Gold accent highlighting
Social Proof

Social Proof Popup

Subtle notifications showing recent activity from other visitors.

File: /components/social-proof-popup.html

Features:

  • Rotates through sample activities (quotes, downloads)
  • Non-intrusive positioning (bottom-left)
  • Auto-dismisses after 6 seconds
  • Progress bar indicator
  • Can be permanently disabled by user
Lead Capture

Calculator Email Capture

Appears after pool cost calculator shows results. Offers to email the detailed breakdown.

File: /components/calculator-email-capture.html

Usage:

showCalculatorEmailCapture({
    lowEstimate: 75000,
    highEstimate: 95000,
    poolType: 'Fiberglass',
    poolSize: '16x32',
    features: ['heating', 'lighting']
});
Lead Capture

Quiz Lead Capture

Gate for pool quiz results. Shows a preview of the recommendation and requires email for full results.

File: /components/quiz-lead-capture.html

Usage:

showQuizLeadCapture({
    recommendedPool: 'Fiberglass Pool',
    matchScore: 95,
    budgetRange: '$60,000 - $100,000',
    priorityFeatures: ['Low Maintenance', 'Quick Install']
}, function(data) {
    // Callback to show full results
    showFullQuizResults(data);
});
Lead Capture

Sticky Mobile CTA

Fixed bottom bar on mobile devices with call and quote buttons.

File: /components/sticky-mobile-cta.html

Features:

  • Only shows on screens 768px or smaller
  • Hides on scroll-down, shows on scroll-up
  • Expandable form for quick quote requests
  • Tracks phone call clicks
  • Respects safe-area-inset for notched devices

Integration Guide

Basic Integration

To add components to a page, include them before the closing </body> tag:

<!-- Before </body> -->

<!-- Seasonal Banner (top of page - move after <nav>) -->
<!-- Copy content from seasonal-banner.html -->

<!-- Exit Intent Popup -->
<!-- Copy content from exit-intent-popup.html -->

<!-- Social Proof Popup -->
<!-- Copy content from social-proof-popup.html -->

<!-- Sticky Mobile CTA -->
<!-- Copy content from sticky-mobile-cta.html -->

</body>

Note: Each component is self-contained. The CSS and JavaScript are included inline to avoid additional HTTP requests. For production, you may want to extract and bundle them.

Lead Data Storage

All components store captured leads in localStorage under the key tsas_leads. Each lead includes:

  • source - Which component captured the lead
  • timestamp - ISO date string
  • Form field data (name, email, phone, etc.)
  • Context data (quiz results, calculator estimates, etc.)

Analytics Events

Components fire Google Analytics events when configured. Make sure gtag is available:

// Event fired on lead capture
gtag('event', 'lead_capture', {
    'event_category': 'Exit Intent',
    'event_label': 'Free Estimate',
    'value': 1
});
